Webbaserad fullstack Bankapplikation

Här byggde jag en webbaserad bankapplikation från grunden. Ramverket som användes var flask, koden var därför i python. Databastekniken var sql med hjälp av docker och dbeaver. Systemet har fullt fungerande login logik med moderna säkerhetssystem. Olika roller har olika rättigheter, inloggad som en admin kan man göra transaktioner mellan konton som går igenom flertalet checkar och sedan ändrar datan i databasen. Detta projekt har unittester för att säkerställa att transaktionslogiken alltid fungerar som den bör. Som man ser på bilderna så var krutet på detta projekt ej frontend, utan att bygga ett funktionellt backendsystem! Projektet landade på drygt 1000 rader kod.
